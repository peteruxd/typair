2. Features Section

Feature 1: Modular Scale Calculator

Title: Mathematical precision, instant results.

Description: Choose from classic musical intervals — Minor Second through Golden Ratio — or enter a custom ratio. TyPair computes every scale step in real time, displaying px, rem, line-height, and letter-spacing for each level from caption to display headings. No more spreadsheets or manual math.

Key Bullet Points:

  • 9 preset ratios plus unlimited custom values
  • Adjustable steps above and below your base size
  • Live computed values in px, rem, line-height, and letter-spacing
  • Fluid clamp() generation for responsive typography out of the box

Alt Text: TyPair scale calculator dropdown showing musical interval options with live computed step values.

Feature 2: Dual Font Controller

Title: Find your perfect pairing in seconds.

Description: Search the full Google Fonts catalog — 1500+ families — with separate selectors for header and body fonts. Each selector previews the font name in its own typeface, and changes apply instantly to the preview canvas. Variable font axes (weight, width, optical size) are supported natively.

Key Bullet Points:

  • Real-time search across 1500+ Google Fonts
  • Independent header and body font controls
  • Variable font axis support (weight, width, optical size)
  • Fonts load on demand with font-display: swap — no bloat, no lag

Alt Text: TyPair Dual Font Controller showing searchable font dropdowns for header and body fonts with live preview.

Feature 3: Lock & Roll

Title: Explore faster. Lock what works, roll through the rest.

Description: Inspired by Coolors, TyPair brings lock-and-roll exploration to font pairing. Found your header font? Lock it. Press Space to cycle through complementary body font options. The curated pairing algorithm ensures serif, sans-serif, display, and mono combinations that actually work together.

Key Bullet Points:

  • Lock either font independently and roll the other
  • Spacebar shortcut for rapid exploration
  • Curated pairing algorithm — serif + sans, display + body, and more
  • Visual lock feedback with smooth font transitions

Alt Text: TyPair Lock & Roll controls showing a locked header font with the roll button cycling body font options.

Feature 4: Live shadcn UI Preview

Title: See your type system on real UI, not abstract blocks.

Description: The right pane defaults to a full page layout built with real shadcn components — navbar, hero section, feature cards, blog excerpt, and footer — all rendered with your selected fonts and scale. Toggle between the full layout view and the detailed hierarchy waterfall. Switch between desktop and mobile viewports to validate responsiveness.

Key Bullet Points:

  • Full shadcn page layout as the default preview
  • Toggle between Full Layout and Hierarchy Waterfall views
  • Desktop (1200px) and Mobile (375px) viewport simulation
  • Custom preview text input for realistic content testing

Alt Text: TyPair preview canvas showing a shadcn UI page layout with navbar, hero, feature cards, and footer rendered in the selected typography.

Feature 5: Responsive Dual Viewport

Title: One scale for desktop. One for mobile. One panel.

Description: Configure independent base sizes and scale ratios for desktop and mobile simultaneously. See both viewports update in real time from a single control panel. No more rebuilding your type system for each breakpoint — TyPair handles both at once.

Key Bullet Points:

  • Independent base size and ratio for desktop and mobile
  • Side-by-side preview frames (1200px + 375px)
  • Optional sync toggle for unified control
  • Fluid typography with configurable min/max viewport widths

Alt Text: TyPair Dual Viewport panel showing separate desktop and mobile scale configurations with live preview frames.

Feature 6: Code Export

Title: From scale to code. One click.

Description: Generate production-ready typography tokens in every format your stack needs. CSS Custom Properties, Tailwind CSS v3 theme.extend, Tailwind v4 @theme, or Style Dictionary JSON — all computed from your active scale. Copy to clipboard with a single click.

Key Bullet Points:

  • Four export formats: CSS, Tailwind v3, Tailwind v4, Style Dictionary JSON
  • One-click copy with visual confirmation
  • Includes font-family, font-size, line-height, and letter-spacing tokens
  • Exports update automatically as you adjust the scale

Alt Text: TyPair Code Export panel showing four format tabs (CSS, Tailwind v3, Tailwind v4, JSON) with copy buttons.

3. About / Positioning Section

Section Title: Built for the way teams really work.

Brand Story: TyPair was born from a simple observation: designers and developers use too many tools to get typography right. A scale calculator here, Google Fonts there, Figma for previews, and manual conversion to CSS. The handoff is slow, error-prone, and frustrating.

We built TyPair to collapse that workflow into a single workspace. One URL. One set of controls. Every format your team needs.

Key Differentiators:

  • **Complete workspace** — Scale, pair, preview, and export in one place. No tab-switching.
  • **Real UI preview** — See your type system on actual shadcn components, not abstract text blocks.
  • **Dual viewport** — Configure desktop and mobile independently. Your scale works everywhere.
  • **Developer-ready export** — CSS, Tailwind v3, Tailwind v4, and JSON from the same scale.
  • **Shareable by design** — Full state encoded in the URL. Share a link, not a screenshot.

Target Keywords: typography scale tool, font pairing, modular scale generator, CSS type scale, Tailwind typography tokens, Google Fonts pairing, design system typography
